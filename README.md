# 中医治则对称模型

## 三病位-治法对称框架

一个基于病位-治法对称性的中医治则公理化模型。通过严格定义三个互斥病位（表、里、半表半里）及其对应的治法，为中医辨证论治提供一套清晰、可复现的决策路径。

## 模型核心

```
病位         治则
──────     ──────
表    →    汗法（辛温/辛甘温/辛凉）
里    →    吐法、下法（涌吐/寒下/温下/润下）
半表半里 →  八纲+五行（虚补实泻、寒热温清 + 补母泻子抑强扶弱；可复合为和法）
```

## 五步决策路径

1. **三病位定位** → 表/里/半表半里
2. **八纲定方向** → 补/泻/温/清
3. **脏腑定靶点** → 心肝脾肺肾
4. **五行定路径** → 补母/泻子/抑强扶弱
5. **六经/三焦/气血津液补充** → 精确化信息

---

## 模型定位

本模型是一个**形式化的辨证决策框架**，解决的是从症状到治法治则之间的路径问题，不涉及治法治则到具体方药的决策——后者取决于医者对药性的理解与临床经验的积累。
