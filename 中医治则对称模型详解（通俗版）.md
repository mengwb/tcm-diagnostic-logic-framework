# 中医治则对称模型详解

## 让中医辨证变简单：一套清晰的三病位决策框架

---

> **声明**：本文作者为中医理论爱好者，非执业中医师。本模型是基于中医经典理论的逻辑重构与形式化整理，旨在为中医学习者提供一套清晰的辨证决策框架。模型本身未经大规模临床验证，不构成诊疗建议。具体临床决策须在执业医师指导下进行。

---

## 前言：为什么要做这套模型？

中医辨证难的症结不在于知识总量过大，而在于**缺乏操作路径**。学生修习了八纲、脏腑、六经、三焦、气血津液五种辨证方法后，面对临床情景时不知道从哪一步开始。五种方法被呈现为平行的选项，但没有给出选择的规则和顺序。

这个模型的核心出发点很简单：**为已有的辨证方法建立一个确定的调用顺序**。

它的核心思想是三句话：
- **病在表** → 汗法
- **病在里（消化道管腔的有形实邪）** → 吐法、下法
- **病在半表半里（脏腑、气血津液、胸腹腔）** → 八纲定方向 + 脏腑定靶点 + 五行定路径（温清消补可复合为和法）

本模型所实现的，是四项具体的、可验证的结构性改进。

---

### 一、模型的核心贡献

#### 1. 为辨证方法建立了调用优先级

本模型提供了一个确定的决策序列：

> **第一步**：确定病位归属（表/里/半表半里）
>
> → 若判定为半表半里：
>
> **第二步**：八纲辨证确定治疗方向（补/泻/温/清）
>
> **第三步**：脏腑辨证确定靶器官（病在何脏何腑）
>
> **第四步**：五行生克确定调控路径（补母/泻子/抑强扶弱）
>
> **第五步**：六经/三焦/气血津液辨证提供补充信息

这一序列的价值不在于发明新的辨证方法，而在于**将已有方法组织成可执行的流程**。不是压缩知识内容，是让知识的调用路径变清晰。

#### 2. 为"里"概念划定了严格边界

"里"是中医理论中负载最重的概念之一。从阳明腑实证到心肾阳虚证，从无形热邪到有形寒积，均曾被归入"里证"范畴。概念的外延过大直接导致其信息量降低——一个能解释一切的概念，实际上丧失了解释力。

本模型将"里"严格限定为**消化道管腔内的有形实邪**（燥屎、宿食、痰涎、虫积等）。这一定义的核心价值不在于完美程度，而在于恢复了"里"这个概念的分辨力——它不再是可以随意填充的容器，而是具有明确边界、可与"表"和"半表半里"形成互斥关系的分析工具。

#### 3. 为半表半里建立了内部结构

传统框架中，半表半里几乎等同于少阳病和和法，其治法范围被人为收窄。本模型将半表半里的治法空间充分打开：八纲定方向、脏腑定靶点、五行定路径——三层递进，使半表半里从"一个治法"扩展为"一个完整的决策子系统"。

半表半里之所以不再显得庞杂无序，是因为它内部有了层级结构：每一步都是上一步的逻辑延续，而非随意组合。

这种秩序的产生有一个关键的逻辑前提：先做了减法。表和里被严格切割之后，半表半里不再是一个被动接收边缘案例的接盘侠——它变成了一个范围大但有内部结构的正空间。在这个空间里，每种辨证方法终于有了自己专属的工作面：八纲不必在同一个里里同时处理下法、清法、温法；脏腑回到了自己的天然主场；五行生克直接落在脏腑靶点上。模糊空间的消失，不是因为每个工具更锋利了，而是因为每个工具终于有了自己的专属工作面。

#### 4. 模型解决的是症状到治法的路径问题

需要明确本模型的边界：它解决的是中医临床决策链中**从症状到治法治则**这一段的问题，不覆盖**从治法治则到具体方药**这一段。后一段取决于医者对药性的理解准确性和临床经验的积累，是任何理论框架无法取代的。

这一边界不是缺陷，而是自知之明。一个框架的价值不在于它覆盖了多少范畴，而在于它在自己承诺覆盖的范畴内是否做到自洽、清晰、无歧义。

#### 5. 统合了各辨证方法的关系

在传统框架中，八纲辨证说"里热"，脏腑辨证说"胃热"，六经辨证说"阳明"——不同辨证体系对同一病证给出了不同表述，学习者难以建立对应关系。在本模型中，这些表述是**同一决策路径上的不同分析层次**，彼此互补而非冲突。

### 二、七大核心优势

**1. 对称性——理论结构的自洽表达**
表→汗法，里→吐下法，半表半里→八纲+五行（可复合为和法）。三病位、三治法一一对应。不仅如此，表证治法的内部分型（辛温/辛甘/辛凉）与里证治法的内部分型（寒下/温下/润下）也呈现结构对称。对称性不是装饰，而是逻辑自洽的外在表征。

**2. 算法化——辨证过程可复现**
从病位定位到治法确定，每一步的输入和输出都有明确定义。初学者不需要依赖直觉或经验积累，只需按路径依次执行即可。辨证从依赖个人悟性的"技艺"转变为步骤明确的"程序"。

**3. 边界清晰——消除概念含混**
每个关键概念都有严格定义：什么是"里"？消化道管腔。什么是"表"？体表。什么是"半表半里"？以上两者之外的全部。三个病位互斥，一个病证在同一时间只属于一个病位。

**4. 辨证方法的协同——消除体系间冲突**
五种辨证方法不再是互相竞争的平行选项，而是同一决策路径上承担不同功能的层次。各居其位，各有其用。

**5. 各方法的角色明确化**

| 辨证方法 | 在模型中的功能 | 调用时机 |
|:---|:---|:---|
| 三病位定位 | 划定分析范围 | 辨证起始 |
| 八纲辨证 | 确定治疗方向 | 病位确定后 |
| 脏腑辨证 | 定位具体靶器官 | 方向确定后 |
| 五行生克 | 设计调控策略 | 靶点确定后 |
| 六经/三焦/气血津液 | 补充细化信息 | 综合判断阶段 |

**6. 概念精确化——消除模糊表达**
"虚则补之"——补什么？虚到什么程度？本模型给出了精确的层次结构：气虚（单纯功能不足）→阳虚（气虚叠加寒象），各有对应的治则和方剂范围。概念不再是可随意解释的术语，而是有边界、有层次、可操作的精确工具。

**7. 定性即公式化——不需要刻度尺**

很多人潜意识里认为"公式化"等于"量化"，但本模型揭示了一个关键认识：**定性本身就是公式化。** 并非只有数字刻度才是公式——定性的方向性判断（"有/无""表/里""虚/实""急/缓"）同样是可复现的公式化操作。

举例说明：两个人同时喊痛，无法绝对量化谁更痛。但在本框架中，不需要这个数字。需要回答的是：这是寒痛还是热痛？虚痛还是实痛？疼痛是表证还是半表半里证？方向定对了，疗效就有保证。定性（定方向）决定了治疗是否有效、是否会让病情加重。方向正确时，刻度只是锦上添花。

模型中出现的"里虚明显""表急深重"等表述，不是模糊的经验描述，而是在定性判断中已占据主导地位的方向性结论。程度判定基于可复现的逻辑定向，而非依赖刻度尺。

这一认识决定了模型的定位：它是**定向层面上的形式化**。方向定了，路径就不会跑偏。

**8. 模型解决的是元问题——不是万能药**

本模型可以类比为数学中的公理与定理体系。公理解决的是元问题——最基本、最单纯的问题。临床中遇到的复杂病证（如表里同病、寒热错杂、多脏腑受累），不是对公理体系的否定，而是多个元问题的复合。

正如复杂数学题需要综合运用多个公理和定理分步骤解题，复杂病证也可通过本模型的元规则进行分层拆解，各元规则互不矛盾、协同运作。数学公理不因问题复杂而失效，恰恰相反——复杂问题需要公理体系才能有序地分解和解决。

**9. 消除例外——理论具有普遍的预测力**
传统中医理论体系的一个深层困境是：理论在指导临床时经常出现例外。同一概念在不同情境下需要不同的解释——"里证"有时用下法，有时用清法；"半表半里"被等同于和法，但大量半表半里的病证并不适用和法。面对这些例外，教学上只能以"特殊"来处理，学习者感受到的是理论对临床的牵强附会，而非自然的推导。

本模型从根本上消除了这个问题。原因很简单：例外产生于定义不严格。一旦每个概念都有了清晰的边界，病位、治法、病性之间的对应关系就是确定的。三病位互斥，三治法各自对应自己的病位，不存在交叉和例外。任何一个病证，无论看起来多么复杂或特殊，都可以在三病位框架内找到唯一的位置和对应的治则方向。

这不是通过增加规则实现的，而是通过**减少规则的模糊性**实现的。模型的预测力不是来自更多的条目，而是来自更清晰的定义。

**8. 不存在边缘案例的可能**——三病位定义互斥且完备，对任一病证依次回答三个问题即可得唯一归属。没有两个病位同时命中或全部落空的情况。边缘案例在逻辑上不可能出现：这不是经验的巧合，而是分类结构本身的必然结果。

---

### 三、理论来源

本模型并非凭空创制，核心框架——将人体疾病按"表、里、半表半里"三个病位分类——源自胡希恕先生的学术思想。胡希恕先生提出"六经来自八纲"，将六经还原为三病位与两病性的组合，这一思路是本模型得以构建的重要基础。

本模型所做的工作，是在胡希恕三病位框架的基础上，对"里"的边界做了进一步限定，并为半表半里建立了一套递进式的内部决策路径。

模型由中医理论爱好者整理，未经临床验证，在学术深度和临床经验上无法与历代医家相比。公布于众的目的是供学习者参考、试用和批评。

## 第一章：三个病位

人体划分为三个互斥的病位：

```
    ┌─────────────┐
    │     表      │  ← 皮肤、肌肉、筋骨
    │  ┌───────┐  │
    │  │ 半表半里│  │  ← 五脏六腑、胸腹腔、气血津液
    │  │ ┌───┐ │  │
    │  │ │ 里 │ │  │  ← 消化道管腔（口→食管→胃→小肠→大肠→肛门）
    │  │ └───┘ │  │
    │  └───────┘  │
    └─────────────┘
```

### 三个病位的定义

| 病位 | 范围 | 举例 |
|:---|:---|:---|
| **表** | 皮肤、肌肉、筋骨、腠理 | 感冒初起、身痛、无汗 |
| **里** | 消化道管腔（口→肛门） | 胃有宿食、肠有燥屎 |
| **半表半里** | 五脏六腑、胸腹腔、气血津液 | 咳嗽、胃痛、肝郁、肾虚、一切内伤杂病 |

**关键点**：三个病位互斥。一个病证在同一时间点只属于一个病位。若同时存在表证和半表半里证，按"先表后里"原则处理。

## 第二章：核心治则映射——三病位对应三治法

```
病位         治则
──────     ──────
表    →    汗法（辛温/辛甘温/辛凉）
里    →    吐法、下法（涌吐/寒下/温下/润下）
半表半里 →  八纲+五行（虚补实泻、寒热温清 + 补母泻子抑强扶弱；可复合为和法）
```

|:---|:---|:---|:---|

| | | 辛甘温解肌 | 桂枝汤 |
| | | 辛凉透表 | 银翘散 |
| **里** | 吐法 | 涌吐 | 瓜蒂散 |
| | 下法 | 寒下 | 大承气汤 |
| | | 温下 | 温脾汤 |
| | | 润下 | 麻子仁丸 |
| **半表半里** | 八纲+五行（可复合为和法） | 虚则补之（补法） | 四君子汤 |
| | | 实则泻之（消法） | 二陈汤 |
| | | 寒者热之（温法） | 四逆汤、理中丸 |
| | | 热者寒之（清法） | 白虎汤、黄连解毒汤 |

## 第三章：虚证与实证的层次结构

### 3.1 基本物质

气、血、津、液是人体最基本生命物质：
- **气**：无形推动力，属阳
- **血、津、液**：有形营养物，属阴

### 3.2 虚证的层次

| 阶段 | 证型 | 公式 | 症状举例 |
|:---|:---|:---|:---|
| 初级阶段 | **气虚** | 气亏虚 | 乏力、气短、自汗（无寒热） |
| 初级阶段 | **阴液亏虚** | 血、津、液亏虚 | 口干、皮肤干、便干（无热象） |
| 高级阶段 | **阳虚** | 气虚 + 寒象 | 乏力 + 畏寒肢冷 |
| 高级阶段 | **阴虚** | 阴液亏虚 + 热象 | 口干 + 五心烦热、盗汗 |

**临床意义**：气虚不等于阳虚。单纯气短乏力但不畏寒者，是气虚而非阳虚。这一区分直接影响用药方向（补气 vs 温阳）。

### 3.3 实证的分类

| 证型 | 公式 | 邪气来源 | 举例 |
|:---|:---|:---|:---|
| **阳盛** | 阳邪亢盛 + 热象 | 功能亢进或外感热邪 | 气分热盛（白虎汤） |
| **阴盛** | 阴邪亢盛 + 寒象 | 痰饮水湿或外感寒邪 | 寒痰阻肺（小青龙汤） |

## 第四章：表证治则（汗法）

病在体表，汗法是唯一出路。

| 类型 | 病性 | 治则 | 代表方 | 关键鉴别点 |
|:---|:---|:---|:---|:---|
| 表实寒（太阳伤寒） | 实寒 | 辛温发汗 | 麻黄汤 | 无汗、恶寒、身痛、脉浮紧 |
| 表虚寒（太阳中风） | 虚寒 | 辛甘温解肌 | 桂枝汤 | 有汗、恶风、脉浮缓 |
| 表热（太阳温病） | 热 | 辛凉透表 | 银翘散 | 发热、微恶寒、咽痛、脉浮数 |

## 第五章：里证治则（吐法、下法）

里证严格限定为消化道管腔的有形实邪。

**吐法**：宿食痰涎停胃 → 瓜蒂散

**下法**：

| 类型 | 病性 | 治则 | 代表方 |
|:---|:---|:---|:---|
| 燥屎热结 | 实热 | 寒下 | 大承气汤 |
| 寒积便秘 | 实寒 | 温下 | 温脾汤 |
| 津亏便秘 | 虚实夹杂 | 润下 | 麻子仁丸 |


**对称性说明**：表证有辛温/辛甘温/辛凉的分型，里证有寒下/温下/润下的分型——两个病位的治法在结构上完全对称。

## 第六章：半表半里——核心决策路径

### 核心规则

半表半里不设单一固定治法。八纲辨证中的补泻温清可根据病证需要灵活组合，这种组合运用在传统上称为"和法"（即小柴胡汤等方剂的治疗原则）。本模型中的和法不是独立的治法，而是温清消补在特定病证（如表里不和、寒热错杂、虚实夹杂）中的复合运用。

> **虚则补之，实则泻之，寒者热之，热者寒之**
>
> **在脏腑层面融入五行生克乘侮（补母、泻子、抑强扶弱）**

### 五步决策路径

| 步骤 | 操作 | 要解决的问题 | 输出 |
|:---|:---|:---|:---|
| 1 | 确认病位 | 病在半表半里？ | 进入半表半里路径 |
| 2 | 八纲定方向 | 虚实寒热？ | 补/泻/温/清 |
| 3 | 脏腑定靶点 | 病在何脏何腑？ | 心/肝/脾/肺/肾 |
| 4 | 五行定路径 | 如何调控？ | 补母/泻子/抑强扶弱 |
| 5 | 补充细化 | 六经/三焦/气血津液？ | 精确化信息 |

### 五行调控三模式

| 模式 | 含义 | 临床案例 |
|:---|:---|:---|
| **虚则补其母** | 子虚时补母以固本 | 肺气虚→补脾（培土生金）→四君子汤 |
| **实则泻其子** | 母实时泻子以疏路 | 肝火旺→泻心火→导赤散 |
| **抑强扶弱** | 乘侮背离时双方同调 | 肝气犯脾→疏肝健脾（抑木扶土）→逍遥散 |

## 第八章：常见问题

### Q1：与胡希恕"六经来自八纲"的关系

胡希恕将六经还原为三病位（表、里、半表半里）和两病性（阴、阳），是本模型的基础。主要差异：
- 胡希恕的"里"包含下法和清法；本模型将"里"限定为消化道有形实邪，无形热邪（白虎汤证）归半表半里
- 胡希恕的半表半里主要对应和法；本模型的半表半里是八纲+五行的综合框架，和法被视为温清消补在寒热虚实夹杂时的复合运用

### Q2：能否取代传统辨证方法？

不能。本模型不发明新方法，只建立调用顺序。工具不变，使用效率提升。

### Q3：表证与半表半里并存时如何处理？

"先表后里"为基本原则。表邪未解而先治里，可能导致邪气内陷。当里虚较甚（如少阴兼表）时可表里同治（麻黄附子细辛汤）。

### Q4：下利清谷的归属

下利清谷是消化功能障碍属脏腑虚寒，非消化道管腔有形实邪，归入**半表半里**（脾肾阳虚），治以温法（四逆汤）。与肠中燥屎的"里证"鉴别要点：有形实邪 vs 功能虚寒。

### Q5：虚实夹杂的处理

原则：急则治标（实邪急迫先泻实），缓则治本（虚证为主先补虚），亦可攻补兼施（如黄龙汤）。具体优先级需结合临床实际，是模型需完善的领域之一。

### Q6：与六经辨证的关系

六经各证型按病位重新分配：太阳→表，阳明腑证→里，阳明经证/少阳/太阴/少阴/厥阴→半表半里。六经的价值在于提示传变路径，而非作为病位分类依据。

### Q7：十枣汤的归属

十枣汤证涉及胸腹水，病位在胸腹腔（半表半里），属水饮内结实证，治以消法（逐水饮）。不属里证。

### Q8：对初学者的价值

提供确定的操作路径：病位→八纲→脏腑→五行→补充，替代漫无头绪的方法选择。

### Q9：是否需要临床验证？

经过10例逻辑验证，全部走通。作为形式化框架，其有效性体现在内部自洽性，而非临床预测力。

## 附：模型的性质、局限与推广

### 模型的性质定位

本模型是一个**形式化的辨证决策框架**，而非临床操作指南。

类比而言：模型提供的是"地图"，呈现了病位、治则和决策路径的整体结构。地图的价值在于帮助使用者建立空间认知、理解各部分之间的逻辑关系。但地图本身不标注每条路的具体走法、所需时间、路况细节——这些属于传统方证学习的内容。

- ✅ 模型提供了结构化的认知框架
- ❌ 模型不替代具体方剂、药量、配伍的学习

这种性质界定是正常的——所有形式化模型的价值都在结构层面。但使用者需明确：本模型需要与传统方证知识配合使用。

### 核心价值总结

**1. 对称性——理论获得结构美**
三病位、三治法对称呼应，表与里的分型也对称。对称性即自洽性的外在表达。

**2. 算法化——辨证过程可复现**
每一步有确定的输入输出，辨证从依赖悟性转变为可执行的流程，降低学习门槛。

**3. 边界清晰——消除概念含混**
"里""表""半表半里"各有严格边界，互斥使用，恢复概念的分辨力。

**4. 体系统一——辨证方法协同工作**
五种辨证方法从平行竞争转为递进协作，各居其位。

**5. 框架开放——可扩展的基础设施**
传统方证、各家学说均可作为补充维度接入，非封闭体系。

**6. 无例外——理论具有普遍的适用性**
传统理论因概念边界模糊，在指导临床时频繁出现例外，不得不以特设性的修补来维持解释力。本模型因定义严格，所有病证均可纳入三病位框架，不存在"特殊"或"例外"的情况。理论的预测力不再依赖于经验性的修补，而是来自逻辑本身的约束。

### 模型边界

1. **本质是形式化框架，不改变临床操作**：本模型不对现有临床实践提出任何修改，仅对辨证知识进行逻辑重组。其有效性体现在内部自洽性和教学辅助价值上。
2. **不覆盖方剂配伍与剂量**：仅解决辨证和治则方向。


### 推广定位建议

**核心定位：教学辅助工具，而非替代性理论体系。**

若定位为"改进版中医理论"，会同时受到经典派和临床派的质疑——前者认为不尊重传统，后者认为不解决具体问题。若定位为"帮助初学者建立辨证思维路径的辅助框架"，则不易招致反对，因为它针对的是公认的教学痛点。

**建议推广路径：**
- 从中医初学者、自学爱好者、住院医师等群体切入
- 以案例演示为主要传播形式
- GitHub仓库作为长期载体
- 长远方向：开发小型决策工具（输入主诉→输出病位判断与治则方向）
